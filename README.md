# electron-example
A simple electron applications

# Features:

-Splash screen

-Html/Css open, close and minimize buttons

-Draggable header

-Works with the latest version of Electron (v17.1.1)

# Before install
Make sure you have installed node.js and git

# How to use
open a (git) terminal and type:
`npm install`
and
`npm test`

This will startup the example project.
